HTML Documentation

Introduction Html or Hyper Text Markup Language is a standard language used by web browsers to interprete images, texts etc into visual or audible web pages. Or in simple terms HTML is a markup language used to build world wide web. a hypertext is a text displayed on a computer, and together with other other texts, a user can access it by a mouse click. Hypertext can contain, tables, forms, lists, images. This format can be vital in sharing information on the web. Html uses tags to characterise text element within a document. This sends instructions to the web browsers on how a document should appear. a pysicist by name Tim Bernes-Lee(known as father of the web) was the one who developed HTML. He wrote the first browser and server software in 1990. By the year 1996, the World Wide Web Consortium(w3c) became the authority to maintain HTML specifications. It became an international standard(ISO) in year 2000. The latest version of HTML is HTML 5. It's faster in web developement. HTML can be used to acces web resources, publish documents online with text, images, lists, tables, etc. It can also be used to create forms to collect user inputs like name, DOB, email, comments etc. Also, it can be used to store users web browser and be able to access it later. It can also be used to web resources such as images and videos. Html has opening tags <> and closing tag <> the closing tag has a slash in between the less and equals to sign.
